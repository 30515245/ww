ZJX复盘的结构与步骤

复盘是一种结构化的回顾方法，通过系统性地回顾目标、评估结果、分析原因和总结经验， 帮助团队或个人从过去的经历中学习，持续改进并提升未来表现。

1. 回顾目标

Goal

  • 明确目标

    回顾项目或活动的初衷、期望达成的结果。

  • 统一认知

    确保团队或个人对最初设定的目标有清晰一致的理解。

关键问题：我们最初想要达成什么？目标是否清晰、可衡量？

2. 评估结果

Result

  • 对比差异

    将实际达到的结果与最初设定的目标进行比较，找出差距。

  • 找出亮点与不足

    记录过程中做得好的地方（成功之处）和做得不好的地方（失败之处）。

关键问题：实际结果与目标有哪些差异？哪些地方超出预期？哪些未达预期？

3. 分析原因

Analysis

  • 探究关键原因

    分析导致成功或失败的根本原因，可以从主观和客观两个层面进行。

  • 识别问题和改进点

    审视过去的做法，找到过程中存在的问题和可以改进的地方。

关键问题：成功或失败的根本原因是什么？哪些因素是可控的？哪些是不可控的？

4. 总结经验

Insight

  • 提炼规律

    将分析出的原因和改进点归纳总结成适用于未来情况的规律和经验。

  • 形成行动计划

    根据总结出的规律，制定下一步的行动计划，包括需要继续、改进或开始的新举措。

关键问题：我们从中学到了什么？未来如何应用这些经验？需要采取哪些具体行动？
