设计逻辑总结

  • 学科闭环延续： 各个周期都保持了“新授 → 运用 → 总结 → 模拟”的闭环。文化课从必修一收尾，衔接高一上下学期预习。信息学从基础算法，迈向竞赛冲刺。英语从词汇积累，过渡到应试运用。
  • 时段功能强化：
    • 第 5、6、7 周围绕“模拟 → 复盘 → 调整”进行，契合竞赛备考节奏。
    • 周末作息起止时间延后，锻炼时长延长，平衡冲刺压力。
    • 周日集中“放松 → 固化 → 适应”，与开学作息同步。
  • 竞赛与校园双重衔接：
    • 竞赛侧： 通过真题模拟、算法模板、设备检查等保障应试能力。
    • 校园侧： 通过作息试运行、模拟早读 / 午休、校园场景训练等降低开学适应成本。
